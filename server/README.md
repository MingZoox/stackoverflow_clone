# Fast create ExpressJs RestApi project

## About init project

- Typescript & Eslint
- User model with 2 roles "admin" & "user"
- Signup & Login with Jsonwebtoken 
- Forgot password with send mail
- Upload avatar to Azure cloud
- CRUD user
